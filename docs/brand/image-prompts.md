# Hope of Glory Ministry — Image Prompt Library

> 30 ready-to-run hyper-realistic image generation prompts, written for **Flux 1.1 Pro** (and cross-compatible with DALL-E 3 and Midjourney v6).
>
> Every prompt below has been built to the rules in `visual-style-guide.md` section 9. Each contains the canonical style block, an explicit aspect ratio, a subject-restriction phrase, and a negative prompt block.

---

## Universal usage notes

- For Flux 1.1 Pro and DALL-E 3: paste the prompt body as written.
- For Midjourney v6: append `--ar 16:9` (or the listed ratio) and `--style raw --v 6.1` flags, and convert the `NEGATIVE:` block to `--no` with comma-separated terms.
- Always render at the highest available resolution (Flux: 2K; DALL-E 3: 1792×1024 or 1024×1024; Midjourney: `--quality 1`).
- Reject any output that contains: visible faces of fictional people, any depiction of Jesus, money, weapons, or political imagery. Regenerate with stronger negative weighting.

## Prompt library governance

- These 30 prompts are the canonical seed set. Variations may be authored, but every new prompt MUST conform to `visual-style-guide.md` section 9.
- The canonical style block must appear verbatim: `photographic, cinematic lighting, reverent, dignified, gold and deep blue palette`.
- A subject-restriction clause must appear: `no people in foreground` OR `silhouetted figures only, no identifiable faces`.
- Aspect ratio must be specified.
- Negative prompt block must be included.
- Any new prompt must be reviewed for the forbidden-imagery list in `visual-style-guide.md` section 9.2 before being added to the library.
