# Corrections

We will get things wrong. When we do, we will say so here.

This page is a public, dated record of corrections to anything we have published — sermons, daily readings, Ask Hope answers, articles, audio, or images. We list what we said, what was wrong with it, and what the correct statement is.

If a piece of content is removed, we say so and why. If a scripture was misquoted, we show the correct text.

> "Buy the truth, and don't sell it. Get wisdom, discipline, and understanding." — Proverbs 23:23 (WEB)

We would rather be corrected than be wrong.

If you have spotted an error — a misquoted verse, a factual mistake, a doctrinal misstep, a broken link, anything — please tell us. You can email **hello@hopeofglory.ministry** or use the [contact page](/contact). We read every message.

We will not list private pastoral conversations or prayer requests here. This page is for public content only.
